About This Project

Copenhagen Court Vision demonstrates how augmented reality can transform municipal decision-making and community engagement. By visualizing proposed improvements in real-world context, citizens can better understand and provide feedback on urban development projects.

This project uses cutting-edge web technologies to create an accessible, cross-platform AR experience that requires no app downloads. The system precisely positions 3D models using QR code calibration points, ensuring accurate 1:1 scale visualization.
